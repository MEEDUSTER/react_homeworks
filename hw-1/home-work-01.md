# Практичне завдання: Створення React-застосунку з компонентною структурою

## Мета завдання

Закріпити знання про компонентну архітектуру React, навчитися створювати та поєднувати окремі компоненти в єдиний застосунок.

## Завдання

1. Створіть новий React-застосунок за допомогою команди `npm create vite@latest` (або використайте існуючий проект).

2. Розробіть React-застосунок з наступними відокремленими компонентами:
   - `Header` - шапка сайту з логотипом та навігаційним меню (мінімум 3 пункти меню).
   - `Sidebar` - бічна панель з додатковою навігацією або корисними посиланнями.
   - `Main` - основний контент сторінки, який повинен містити заголовок, декілька параграфів тексту та хоча б одне зображення.
   - `Footer` - підвал сайту з інформацією про авторські права та контактними даними.

3. Всі компоненти повинні бути створені як окремі файли в папці `components`.

4. Імпортуйте та використайте всі компоненти в головному компоненті `App.ts`.

5. Додайте базову стилізацію для кожного компонента за допомогою CSS:
   - Використайте різні кольори фону для кожного компонента.
   - Додайте відступи та рамки для візуального розділення компонентів.

## Критерії оцінювання

- Правильність структури компонентів та їх взаємодії.
- Коректне використання props для передачі даних.
- Якість та чистота коду (назви змінних, структура файлів).
- Естетичність та зручність інтерфейсу.

## Термін виконання

10 днів з моменту отримання завдання.

## Форма здачі

Посилання на GitHub репозиторій з вашим проектом та скріншоти готового застосунку.
